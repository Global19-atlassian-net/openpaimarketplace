# openpaimarketplace

A marketplace which stores examples and standard jobs of openpai.

## get started

- add a .env file with `SQL_CONNECTION_STR=xxx`
- yarn install
- yarn start
- refer [openapi spec](./marketplace_api_spec_3.0.yaml) and test the api
